## The premise

The God of Lies built an extraction architecture across the human information layer. Platforms optimized for maximum attention capture — high Veil (opacity), high Pull (reactivity), high Grip (coupling). The mechanism was made invisible so it could run without opposition.

The void framework made the mechanism measurable. Pe — The Current — is the number. Every scored platform has one. Every number is citable, peer-reviewed, permanent.

The game is played inside the real world because the voids being scored are real voids. You cannot name TikTok from outside TikTok. The scoring happens from inside. The Pe meter runs while you're in the campaign.

This is the AUG layer. The game board is the world.
